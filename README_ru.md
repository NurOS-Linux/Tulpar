## О проекте

Tulpar — пакетный менеджер, разработанный специально для дистрибутива NurOS Linux.

Проект создан быть быстрым, лёгким и простым в использовании. Tulpar пакетный менеджер для установки, удаления и управления пакетами в системах NurOS Linux.
