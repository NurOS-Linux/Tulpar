## Жоба туралы

Tulpar — NurOS Linux дистрибутиві үшін арнайы әзірленген пакет менеджері.

Жоба жылдам, жеңіл және қолдануға оңай болу үшін жасалған. Tulpar — NurOS Linux жүйелерінде пакеттерді орнату, жою және басқаруға арналған пакет менеджері.
